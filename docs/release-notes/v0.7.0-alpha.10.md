# Fantasy Earth 0.7.0-alpha.10

## Naming migration

- Canonical public name: `Fantasy Earth / 魔幻地球`
- Canonical mod id and resource namespace: `fantasy_earth`
- Canonical repository and artifact prefix: `fantasy-earth`
- Canonical Java package: `com.xxsx.fantasyearth`
- Legacy `earth_online_magic:*` block, item, entity, block-entity, menu and feature ids resolve through NeoForge registry aliases.
- Legacy `earth_online_arcana.*` player data migrates to `earth_arcana.*` on first access.

## Target

- Minecraft 26.2
- NeoForge 26.2.0.7-beta
- JDK 25
- License: `AGPL-3.0-only`
- Artifact: `fantasy-earth-neoforge-26.2-0.7.0-alpha.10.jar`

This is an in-development prerelease. Build and static migration checks are complete; loading an existing alpha.9 world remains a runtime test item.
