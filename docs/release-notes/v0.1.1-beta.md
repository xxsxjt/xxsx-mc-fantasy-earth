# Earth Online: Magic v0.1.1-beta

开发中测试版 / in-development beta.

## Target

- Minecraft: 26.2
- Loader: NeoForge 26.2.0.7-beta
- Java: 25
- Mod id: `earth_online_magic`
- Jar: `earth-online-magic-neoforge-26.2-0.1.1.jar`
- SHA256: `D7DF004283CA3D76E057D58DE1ED5AEAF1D011CC10C0721303D7D41B44477A0D`

## Highlights

- Adds the first chunk-level aether field runtime.
- Aether crystal clusters, ritual/enchanting structures, rune/book structures and alchemy/energy materials now contribute to local magic readings.
- Magic attunement now has cooldown and disturbs the local chunk instead of being a free repeat-click recovery loop.
- Field notebook, initiation notes, adaptation notes and magic blocks show readable aether source and disturbance feedback.
- Updates bilingual Chinese / English text for field readings, cooldowns and player guidance.

## Verification

- Built locally with `.\gradlew.bat build --no-daemon --offline`.
- Deployed to a NeoForge 26.2.0.7-beta client instance.
- Deployed jar hash matched the local build artifact.
- Jar contents were inspected for the new field classes and language resources.

## Known Limits

- Runtime smoke testing is still ongoing.
- Cross-chunk diffusion, persistent source registration, mana salt / crystal recovery, ritual multiblocks and handbook GUI field pages are not implemented yet.
- Worldgen and field values may change between beta builds.
