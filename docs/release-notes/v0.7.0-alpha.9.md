# Fantasy Earth on Minecraft 0.7.0-alpha.9

Development test build for Minecraft 26.2 / NeoForge 26.2.0.7-beta.

- Spreads settlement resident and familiar creation across server ticks instead of creating a full settlement synchronously after the 40-tick warmup.
- Limits all magic settlement entity additions to two per server tick.
- Defers settlement initialization until a player is within 48 blocks, avoiding work for distant loaded anchors.
- Persists `SpawnIndex`, so chunk unloads and server restarts resume without restarting the whole settlement.
- Keeps the alpha.8 transparent logo and mana-wisp flying-speed fix.

Artifact: `fantasy-earth-on-minecraft-neoforge-26.2-0.7.0-alpha.9.jar`

Verification limit: source, metadata and complete offline build checks pass; a live multi-anchor load test still requires an explicitly authorized client/server smoke test.
