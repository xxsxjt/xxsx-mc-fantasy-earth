# Codex Notes For Fantasy Earth on Minecraft

- This is the independent repository for `Fantasy Earth on Minecraft / 魔幻地球 on Minecraft`.
- Do not merge gameplay code into `earth_on_minecraft` or sibling mods unless explicitly requested.
- Keep the mod independently runnable. Integrate `earth_on_minecraft`, `earth_human`, and `earth_online_xuanhuan` only through optional compatibility layers.
- Preserve the technical mod id `earth_online_magic` unless a deliberate save-breaking migration is requested.
- Target line is expected to follow `NeoForge 26.2 / MC 26.2 / JDK 25` unless the user changes it.
- This mod recreates broadly recognized fantasy systems and content, not real survival difficulty.
